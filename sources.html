Comandos Linux

Comandos esenciales de administración del sistema Linux

Linux Terminal

Listado de comandos útiles para la administración de sistemas Linux.

Gestión de archivos y directorios:

# Listar el contenido de un directorio
ls
ls -a

# Cambiar el directorio actual
cd

# Mostrar la ruta completa del directorio de trabajo actual
pwd

# Crear una carpeta nueva
mkdir nombre_carpeta

# Eliminar una carpeta vacía
rmdir nombre_carpeta

# Eliminar archivos
rm -r nombre_archivo

# Copiar archivos y directorios
cp origen destino

# Mover archivos y directorios
mv origen destino

# Crear un archivo vacío
touch ejemplo.txt

# Mostrar el contenido de un archivo
cat nombre_archivo

# Buscar archivos y directorios según varios criterios
find /ruta/busqueda -name "nombre_archivo"

# Cambiar permisos de archivo
chmod 755 nombre_archivo

# Comprimir directorio en tar y zip
tar -czvf ejemplo.tar.gz carpeta_destino/
zip -r ejemplo.zip carpeta_destino/

# Descomprimir directorio
tar -xvf ejemplo.tar.gz
unzip ejemplo.zip

Información del sistema y procesos:

# Mostrar los procesos que se están ejecutando y el uso de los recursos del sistema en tiempo real (CPU, memoria)
top

# Mostrar una instantánea de los procesos en ejecución
ps

# Mostrar el uso del espacio en disco
df
du

# Mostrar el uso de memoria del sistema
free

# Mostrar cuánto tiempo lleva el sistema en funcionamiento
uptime

Redes y seguridad:

# Conectarse de forma segura y remota a otro equipo
ssh

# Mostrar o configurar las interfaces de red
ifconfig / ip

Comandos explicados:

  • df -h: Muestra el uso de disco en formato legible
  • htop: Monitor de procesos interactivo (necesita instalación)

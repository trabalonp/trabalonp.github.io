API Python

Creación de APIs REST con Flask

Python Flask API REST

Crear una API REST simple utilizando Flask para gestionar una lista de tareas. Este ejemplo muestra cómo implementar operaciones CRUD básicas.

from flask import Flask, jsonify, request

app = Flask(__name__)

tasks = []

@app.route('/tasks', methods=['GET'])
def get_tasks():
    return jsonify(tasks)

@app.route('/tasks', methods=['POST'])
def add_task():
    task = request.json
    tasks.append(task)
    return jsonify(task), 201

@app.route('/tasks/<int:task_id>', methods=['DELETE'])
def delete_task(task_id):
    if task_id < len(tasks):
        deleted_task = tasks.pop(task_id)
        return jsonify(deleted_task)
    return jsonify({'error': 'Task not found'}), 404

if __name__ == '__main__':
    app.run(debug=True)

Explicación del código:

  • Flask: Framework web minimalista para Python
  • GET /tasks: Obtiene todas las tareas
  • POST /tasks: Crea una nueva tarea
  • DELETE /tasks/{id}: Elimina una tarea específica

Para ejecutar este código, necesitas instalar Flask:

pip install flask

Comandos Linux

Comandos esenciales de administración

Linux Terminal Bash

Lista de comandos útiles para la administración de sistemas Linux que uso frecuentemente.

# Encontrar archivos por nombre
find /ruta/busqueda -name "nombre_archivo"

# Buscar texto dentro de archivos
grep -r "texto_a_buscar" /ruta/

# Ver uso de disco
df -h

# Ver uso de memoria
free -h

# Monitorear procesos en tiempo real
htop

# Cambiar permisos de archivo
chmod 755 archivo.txt

# Cambiar propietario de archivo
chown usuario:grupo archivo.txt

# Comprimir directorio
tar -czvf archivo_comprimido.tar.gz directorio/

# Descomprimir archivo
tar -xzvf archivo_comprimido.tar.gz

Comandos explicados:

  • find: Busca archivos en el sistema de archivos
  • grep: Busca patrones de texto dentro de archivos
  • df -h: Muestra el uso de disco en formato legible
  • htop: Monitor de procesos interactivo (necesita instalación)
